폼 관련 태그

form 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그이다.
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다.
* action 속성: 폼의 입력된 값들을 전송받을 서버의 주소를 입력한다.
* method 속성: get / post 방식으로 전송 방식을 지정한다. (Http method 방식 지정)

 

submit 버튼을 눌렀을 때 action에 지정한 경로로 method에 지정한 방식에 따라
input 태그에 입력한 값들을 전달한다.

필드셋의 제목을 작성하는 부분이다.


필드셋으로 묶은 영역별로 구분된다.


text와 관련된 input 태그들

type="text"

- 한 줄짜리 텍스트를 입력할 수 있는 텍스트 상자이다.



type="password"

- 비밀번호를 입력할 수 있는 텍스트 상자이다.



type="search", type="url", type="email", type="tel"

- 겉모습은 type="text"와 비슷하지만 각각의 정보에 맞게 분화된 기능을 제공하는 텍스트 상자이다.
